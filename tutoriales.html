Como Instalar Mono en Raspbian Wheezy 

							
mkdir ~/monosource
cd ~/monosource

En este sitio busca la version mas reciente
http://download.mono-project.com/sources/mono/

Para el momento es la 4.02

wget http://download.mono-project.com/sources/mono/mono-4.0.2.5.tar.bz2

Descomprime
tar -xjvf mono-4.0.2.5.tar.bz2

Instala prerequisitos							
sudo apt-get install build-essential autoconf automake binutils libtool libglib2.0-dev libxrender-dev libfontconfig1-dev libpng12-dev libgif-dev libjpeg8-dev libtiff5-dev libexif-dev gettext libcairo2-dev htop							

Configura e instala, puede demorar..
./configure --prefix=/usr/local; sudo SKIP_AOT=true make; sudo SKIP_AOT=true make install;

sudo reboot

Comprueba la version de la instalacion
mono -V
